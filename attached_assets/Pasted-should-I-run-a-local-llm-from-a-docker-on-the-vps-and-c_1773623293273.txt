4. Easiest way to deploy (10 minutes)
SSH into VPS and run:

sudo apt update
sudo apt install docker.io docker-compose -y
sudo systemctl enable docker
Install Ollama
docker run -d \
--name ollama \
-p 11434:11434 \
-v ollama:/root/.ollama \
ollama/ollama
Install ChatGPT-style UI
docker run -d \
-p 3000:8080 \
--name openwebui \
-v open-webui:/app/backend/data \
ghcr.io/open-webui/open-webui:main
Now open in browser:

http://YOUR-VPS-IP:3000
You now have your own ChatGPT server.

5. Install models
Inside the VPS:

docker exec -it ollama bash
Then:

ollama pull llama3
ollama pull mistral
ollama pull deepseek-coder
ollama pull codellama
6. Access it from anywhere
Your API endpoint becomes:

http://YOUR-VPS-IP:11434

curl http://YOUR-VPS-IP:11434/api/generate \

7. Secure access (important)
Do NOT leave it fully open.

Best options:

Option 1 (recommended)
Use Cloudflare Tunnel

cloudflared tunnel
Then your LLM becomes

https://ai.yourdomain.com
Option 2
Restrict by IP with firewall.

ufw allow from YOUR_IP to any port 11434
